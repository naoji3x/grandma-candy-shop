# 業務イベント一覧

業務上の重要なイベント（販売、仕入、在庫調整など）を一覧として定義します。

## ドキュメントの一覧

| ドキュメント     | 略称 | 内容                                                          | 関連テンプレート                                                                                               |
| ---------------- | ---- | ------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------- |
| 業務イベント一覧 | BEL  | 業務イベントの一覧と、イベントが表す事実を定義する            | [ルール](../../../handbook/rules/bel-rules.md) / [作成指示](../../../handbook/instructions/bel-instruction.md) |
| 業務イベント仕様 | BEV  | 個別イベントの詳細（イベントの事実データ/条件など）を定義する | [ルール](../../../handbook/rules/bev-rules.md) / [作成指示](../../../handbook/instructions/bev-instruction.md) |

## 記法ガイド

1. 一覧（BEL）を作成・更新する場合は [bel-rules.md](../../../handbook/rules/bel-rules.md) を参照してください。
2. 個別仕様（BEV）を作成・更新する場合は [bev-rules.md](../../../handbook/rules/bev-rules.md) を参照してください。
3. 生成AIを活用する場合は [bel-instruction.md](../../../handbook/instructions/bel-instruction.md) / [bev-instruction.md](../../../handbook/instructions/bev-instruction.md) をプロンプトのテンプレートとして利用してください。
